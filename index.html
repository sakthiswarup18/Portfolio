From Data to Insights: Examining Fatal Police Shootings in the U.S.

UMass Dartmouth | Sept 2023 - Dec 2024

This project analyzes fatal police shootings in the United States, identifying patterns and risk factors based on demographic, geographic, and situational variables. Using K-Means Clustering and Logistic Regression, the study explores racial and gender disparities, geographic trends, and predictive modeling of incident scenarios.

Key Findings:
  • Higher incidence among white, black, and Hispanic populations, with males being disproportionately affected.
  • Geographic concentration in urban areas such as Los Angeles, Houston, and New York, with lower occurrences in rural regions.
  • K-Means clustering identified spatial hotspots in California, Texas, and Florida where shootings are more frequent.
  • Logistic Regression predicted gun-related incidents accurately but struggled with cases involving mental illness or body camera presence.
Data Sources & Tools:
  • Dataset from The Washington Post covering fatal police shootings since 2015.
  • Geospatial analysis using Folium for mapping incident locations.
  • Machine Learning models implemented in Python (Scikit-learn, Pandas, Matplotlib, Seaborn).

Detecting Credit Card Fraud Using Machine Learning

UMass Dartmouth | Sept 2024 - Present

This project focuses on developing and evaluating machine learning models to detect fraudulent credit card transactions. Using LightGBM, XGBoost, CatBoost, and Random Forest, we built a robust fraud detection system capable of identifying fraudulent transactions with high accuracy.

Key Features:
  • Dataset of 284,807 transactions, with only 0.17% fraudulent cases.
  • Feature engineering techniques to enhance model accuracy.
  • Exploratory Data Analysis (EDA) to uncover hidden patterns.
  • Implementation of multiple classifiers (LightGBM, XGBoost, CatBoost, Random Forest).
  • Performance evaluation using AUC score and confusion matrix.
Results & Impact:
  • LightGBM achieved a test AUC of 0.93, outperforming other models.
  • XGBoost delivered a validation AUC of 0.984, indicating strong predictive power.
  • Feature importance analysis identified V14, V10, V12, V17, and Amount as key predictors.
  • Developed an interpretable fraud detection system for real-world applications.
Future Enhancements:
  • Integration of real-time fraud detection for live transactions.

Personalized AI-Driven Cybersecurity Response System for IoT Devices

Published | University of Massachusetts Dartmouth

This research explores a new AI-driven cybersecurity approach for IoT devices, focusing on personalized security responses based on user behavior. Traditional security models struggle with high false positives and evolving cyber threats, making them inefficient in dynamic IoT environments. Our study proposes an adaptive, behavior-based security system that learns individual usage patterns to detect anomalies and respond to threats in real time.

Key Highlights:
  • Analyzes limitations of traditional rule-based security models for IoT.
  • Introduces an AI-driven security system that adapts to unique user behaviors.
  • Utilizes behavioral modeling, anomaly detection, and federated learning.
  • Reduces false alarms while increasing threat detection accuracy.
  • Explores privacy-preserving techniques through on-device AI learning.
Impact and Applications:
  • Enhances real-time cybersecurity in smart homes, healthcare, and industrial IoT.
  • Reduces over-reliance on predefined threat databases, improving detection accuracy.
  • Provides scalable, adaptive security for billions of IoT devices.
Future Research Directions:
  • Developing lightweight AI models for low-power IoT devices.
  • Integrating blockchain-based authentication for decentralized security.
  • Refining federated learning techniques for privacy-preserving model training.
